# imageAsymmetryAnalysis
## Tool used in ARKS Paper VI (Asymmetries &amp; Offsets). 

The code includes the scripts to reproduce figures C1, C2, C3, C4 and C5 from ARKS VI.
These figures are the self-subtraction profiles, for the rotation, mirrored minor axis, and mirrored major axis.

All data for the ARKS project is published open source in the ARKS Dataverse at following location: https://dataverse.harvard.edu/dataverse/arkslp

Original code adapted from 'rotate_subtract.py' by G. M. Kennedy, stored here: https://github.com/SebaMarino/ARKS/tree/master/analysis_scripts

non-standard installs needed:
- cmocean, seaborn, pandas, scipy, json, astropy
